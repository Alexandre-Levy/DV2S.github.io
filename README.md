# D-V2S Project Website

This repository hosts the project page for the paper:

> **From Driving Videos to Simulatable Scenarios**  
> Alexandre Levy, Ernest Valveny Llobet, Antonio Manuel Lopez Peña  
> ITSC 2026

**Live site**: https://alexandre-levy.github.io/DV2S.github.io

## Updating the page

- Replace placeholder images in `static/images/` with real figures
- Drop the paper PDF into `static/pdfs/DV2S.pdf`
- Update result images in the Qualitative Results section of `index.html`

## GitHub Pages setup

In the repo settings → Pages → set source to **Deploy from a branch**, branch `main`, folder `/` (root).
